Doggy is a web application that allows users to upload a photo of a dog and receive an AI-powered breed identification, along with basic hints about care and lifestyle. It is developed as part of a University Software Engineering project work, using React, TypeScript, Vite, Tailwind CSS and shadcn/ui, following the template and guidelines provided by the course. 

--- 

## Project Idea 
Doggy aims to provide a simple and intuitive interface where users can upload a dog photo and obtain a breed prediction.

This **initial stage** focuses only on: 
- preparing the repository structure 
- defining the documentation skeleton 
- outlining the high-level idea 
- following the official project work checklist 

No functional implementation is included yet.
